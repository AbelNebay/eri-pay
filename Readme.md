# ERI Pay – Digital Payment System

## Overview
A mobile payment system similar to M-Pesa designed for digital transactions using Eritrea’s telecom infrastructure.

## Features
- Send and receive money
- Mobile wallet system
- Transaction history
- Secure authentication

## Tech Stack
- Java (Android)
- Telecom API integration
- Secure backend system

## Use Case
- Mobile banking
- Digital payments
- Financial inclusion
